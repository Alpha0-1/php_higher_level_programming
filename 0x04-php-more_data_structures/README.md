# 0x04 - PHP: More Data Structures

This directory contains PHP scripts demonstrating various array and associative array manipulations in PHP. These scripts aim to deepen understanding of PHP data structures by providing practical implementations of common tasks.

## Files

| File | Description |
|------|-------------|
| `0-square_matrix_simple.php` | Creates a square matrix with squared values |
| `1-search_replace.php` | Replaces all occurrences of a value in an array |
| `2-uniq_add.php` | Sums unique integers in an array |
| `3-common_elements.php` | Finds elements common to two arrays |
| `4-only_diff_elements.php` | Finds elements present in only one of the arrays |
| `5-number_keys.php` | Counts keys in an array |
| `6-print_sorted_array.php` | Prints a sorted array |
| `7-update_array.php` | Updates a value in an associative array |
| `8-simple_delete.php` | Deletes a key from an associative array |
| `9-multiply_by_2.php` | Multiplies all numeric values in an array by 2 |
| `10-best_score.php` | Finds the highest score in an associative array |
| `11-multiply_array_map.php` | Uses `array_map()` to multiply each value by 2 |
| `12-roman_to_int.php` | Converts Roman numerals to integers |
| `100-weight_average.php` | Calculates the weighted average |
| `101-square_matrix_map.php` | Creates a square matrix using `array_map()` |
| `102-complex_delete.php` | Deletes multiple keys from an associative array |
| `103-advanced_sort.php` | Sorts an associative array by value or key |

Each file contains example usage and comments to help understand the underlying logic.
